How it works · From scan to unlimited visuals

Solaya collapses what used to require studios, photographers, and external 3D vendors into a single scan on the brand side, with the entire pipeline automated on Google Cloud.

1 · Scan / Capture
A merchandiser or content team member performs an approximately 3-minute handheld capture around the product, optionally flipping it to reveal the bottom. Raw images and metadata are securely written to Cloud Storage, where basic color and noise corrections are applied.
2 · Process / Reconstruct
Vertex AI orchestrates a multi-stage inference pipeline: multi-view geometry estimation, dense point cloud, mesh reconstruction, and physically-aware texture generation. Every output is quality-scored to meet e-commerce visual standards for detail, sharpness, and consistency.
3 · Redeem / Publish
Generated 3D models, stills, and videos are exposed through Cloud Run / GKE APIs and a management console, powering product pages, social media campaigns, interactive 3D viewers, virtual try-on, and continuous creative testing— effectively “unlimited angles, unlimited styles” from a single capture.

Deep integration with Google Cloud

Solaya uses a cloud-native architecture where 3D GenAI training, inference, and content delivery run entirely on Google Cloud, combining scalability, performance, and enterprise-grade security.

Core products

Storage: Cloud Storage for scans and generated assets
AI / ML: Vertex AI for training and batch / online inference
Compute: Cloud Run + GKE for rendering and API services
Data: BigQuery for asset indexing, performance analytics, and content insights
Observability: Cloud Logging & Cloud Monitoring for full pipeline visibility

Security is enforced with IAM for access control, VPC Service Controls for tenant isolation, and CMEK via Cloud KMS for encryption—ensuring brands retain ownership and control of all generated 3D assets and visual content.
