工作流 · 从手机扫描到无限视觉内容

Solaya 把传统需要摄影棚、摄影师与 3D 外包的流程，收敛成用户侧的一次简单扫描， 其余全部托管在 Google Cloud 上的自动化 pipeline 中。

1 · Scan / Capture
用户使用手机对产品进行约 3 分钟环绕拍摄，可视需求翻转产品以捕获底部。原始图像与元数据被安全写入 Cloud Storage，并进行基础校正与去噪。
2 · Process / Reconstruct
由 Vertex AI 启动多阶段推理：多视角几何估计、稠密点云生成、网格重建与纹理贴图。管线对输出质量进行自动评分与优化，以满足电商级别视觉标准。
3 · Redeem / Publish
生成的 3D 模型、图片与视频通过 Cloud Run / GKE 暴露为 API 与管理后台，支持电商页面、社交媒体营销、虚拟试戴与互动展示，实现真正“无限角度、无限风格”的内容扩展。

与 Google Cloud 的深度集成

Solaya 采用云原生架构，将 3D GenAI 的训练、推理与内容交付全部落在 Google Cloud 上，既具备强扩展性，也满足企业级安全与合规要求。

核心产品

Storage：Cloud Storage 存储扫描图像与输出资产
AI / ML：Vertex AI 完成训练与批量 / 实时推理
Compute：Cloud Run + GKE 运行渲染与 API 服务
Data：BigQuery 用于资产索引与内容分析
Observability：Cloud Logging & Monitoring 全链路监控

安全层面，平台通过 IAM 控制访问权限，结合 VPC Service Controls 与 Cloud KMS（CMEK）实现多租户隔离与端到端加密，满足品牌方对数据主权与资产所有权的要求。
